# 安易网络安全意识评测系统 (AnYi-Security-Awareness)
基于深度知识追踪（DKT）的网络安全意识评测系统，通过模拟真实钓鱼攻击场景，客观采集用户无意识行为数据，智能评估用户网络安全意识等级并提供个性化建议。

## 项目简介
随着网络攻击日益频繁，人员安全意识成为网络安全最薄弱的环节。传统问卷调查无法真实反映用户在威胁场景下的无意识行为，而本系统通过模拟钓鱼邮件、钓鱼网页等攻击场景，采集用户真实操作数据，结合深度知识追踪模型（DKT）与分类评级机制，实现对用户网络安全意识水平的精准评估。

系统已在 GitHub 开源，代码结构清晰，文档齐全，适用于企业安全培训、高校网络安全实训等场景。

## 功能特点
**攻击情景模拟**：支持邮件钓鱼、网页钓鱼等多类型攻击模板配置，真实还原攻击场景。

**行为数据采集**：自动监听用户打开邮件、点击链接、提交密码、扫描二维码等关键行为，数据客观可信。

**智能意识评估**：基于深度知识追踪（DKT）模型，对用户行为序列建模，预测行为倾向并分类意识等级（优秀/良好/及格/差）。

**可视化报表**：提供行为统计图表、用户等级分布、缺陷分析与改进建议。

**多角色权限**：区分用户管理员、数据获取员、数据评估员、受测者四种角色，分工明确。

**开源可扩展**：前后端分离架构，模块化设计，支持二次开发。

## 快速开始

### 环境要求

- Python 3.8+
- Node.js 14+
- MySQL 5.7+
- PyTorch 1.9+

### 安装步骤

**克隆仓库**
   ```bash
   git clone https://github.com/GAOBW123/AnYi-Security-Awareness.git
   cd AnYi-Security-Awareness
```
### 访问系统
前端控制台：http://localhost:3000
后端API：http://localhost:5000
